Основное меню

В центре экрана расположен список доступных разделов. В его составе необходимо отметить следующие пункты.

  • Литература. Данный пункт содержит несколько подпунктов, каждый из которых содержит список литературы по одной главе Красной книги.
  • Приложение к Красной книге. В этом пункте содержатся оригинальные приложения к Красной книге: "Животные, исключенные из Красной книги", "Исчезнувшая в России мировая фауна", "Животные, нуждающиеся в особом внимании".
  • Чтобы из списка подпунктов снова выйти в список основных разделов, необходимо щелкнуть ссылку "Оглавление".
  • Информация по охраняемым видам. Этот пункт предоставляет доступ к основному инструменту, входящему в состав программного пакета - интерфейсу поиска и выбора видов, перечисленных в Красной книге.
