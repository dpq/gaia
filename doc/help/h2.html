Поиск и выбор вида

Основную площадь этого окна занимают систематический и алфавитный списки видов.

  • Слева расположен список глав Красной книги. Выберите интересующую главу щелчком по соответствующему переключателю, чтобы загрузить в систематический список и в алфавитный указатель данные по этой главе.
  • Систематический список имеет древовидную структуру, что позволяет минимизировать таксономические единицы, не представляющие интереса.
  • Алфавитный указатель видов может работать как в режиме сортировки по латинским названиям, так и по русским. Переключение между этими двумя режимами производится щелчком по соответствующей кнопке (расположены вверху указателя)
  • Чтобы вернуться в список разделов, щелкните по кнопке "Оглавление" слева вверху.
  • Если подвести указатель мыши к названию вида в алфавитном указателе, то над ним высветится латинское название вида, если указатель работает в режиме русских названий, и наоборот.
  • Если подвести указатель мыши к строке с информацией по виду в систематическом списке, то над ним высветится полная информация по виду. Сделано это для того, чтобы можно было просматривать строки настолько длинные, что они не помещаются в стандартную ширину списка, и чтобы не принуждать пользователя работать с горизонтальной прокруткой списка.
  • Двойным щелчком по строке с именем вида в любом из списков осуществляется переход в окно просмотра информации по конкретному виду.
